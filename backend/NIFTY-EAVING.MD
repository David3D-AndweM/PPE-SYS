## Key Design Decisions

| Concern | Decision |
|---|---|
| JWT claims | Embed roles + employee_id + site_ids â†’ no per-request DB lookup for authorization |
| QR payload | HMAC-signed, not raw UUID â†’ forgery prevention |
| Approval levels | JSONB field on PPEConfiguration â†’ add levels with zero code change |
| PPE config scope | dept > site > system (most specific wins) |
| Stock deduction timing | On `finalize_issue()`, not on approval â†’ matches physical reality |
| ASGI server | Daphne or Uvicorn workers (Gunicorn alone breaks Channels) |
| Celery beat scheduler | `DatabaseScheduler` â†’ schedule managed via Django admin |
| State management (Flutter) | BLoC â€” testable, predictable across 5 role UIs |
